# Practical Statistics for Data Scientists

Repository ini berisi reproduksi kode dan rangkuman dari buku *Practical Statistics for Data Scientists (O'Reilly)*.

## Tujuan
Memahami konsep statistik dasar yang digunakan dalam data science melalui implementasi Python dan visualisasi.

## Struktur Repository

### Chapter 1 – Introduction
Pengantar statistik dalam data science dan pentingnya eksplorasi data.

### Chapter 2 – Data and Sampling
Konsep populasi, sampel, dan teknik sampling.

### Chapter 3 – Statistical Experiments
Eksperimen statistik dan pengujian hipotesis.

### Chapter 4 – Sampling and Bias
Bias dalam data dan bagaimana sampling mempengaruhi hasil analisis.

### Chapter 5 – Statistical Inference
Estimasi dan confidence interval untuk menarik kesimpulan dari data.

### Chapter 6 – Regression
Hubungan antar variabel menggunakan model regresi.

### Chapter 7 – Clustering
Pengelompokan data menggunakan metode unsupervised learning.

## Kesimpulan
Statistik merupakan dasar penting dalam data science untuk memahami data, membuat model, dan menarik kesimpulan yang akurat.
